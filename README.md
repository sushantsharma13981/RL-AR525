## Repository Overview
This repository contains course projects and assignments for AR 525, focusing on applying reinforcement learning techniques to robotic systems. The projects explore various RL algorithms in the context of robotic navigation, manipulation, and control.



**Technologies:** Python, PyBullet, NumPy, Matplotlib, Seaborn

## Getting Started

### Prerequisites
- Python 3.10 or 3.11
- C++ Build Tools (For Windows)

### Setup
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd AR525-RL
   ```

2. Navigate to specific assignment:
   ```bash
   cd a1
   pip install -r requirements.txt
   python main.py
   ```

## Assignments Overview

### Assignment 1: Dynamic Programming for Grid Navigation
- **Objective**: Implement Policy Evaluation, Policy Iteration, and Value Iteration
- **Environment**: 5x6 grid with UR5 robot and obstacles
- **Deliverables**: Code implementation, simulation demo, analysis report

### Assignment 2: Model-Free Control for Drone Hovering
- **Objective**: Implement Monte Carlo Control and Q-Learning to stabilize a drone at a target hover position
- **Environment**: PyBullet drone simulator with discretized 3D state and thrust actions
- **Deliverables**: RL implementations, training/evaluation scripts, visualizations/analysis of learned policy
